# Debojeet — Portfolio

A luxury minimalist portfolio built with a cinematic scroll-driven hero animation, glassmorphism UI, and a dark carbon-black aesthetic.

## 🔗 Live Demo

> _Coming soon — deploy on GitHub Pages or Vercel_

## ✨ Features

- **Cinematic Hero** — 240-frame scroll-driven canvas animation
- **Glassmorphism UI** — Frosted-glass buttons and cards
- **Infinite Marquee** — Colorful tech stack showcase
- **Responsive Design** — Desktop-first with mobile support
- **Mailto CTA** — Direct email contact via Gmail

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript
- Tailwind CSS (CDN)
- Font Awesome Icons
- Google Fonts (Manrope, Inter)

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/notDeboatall/portfolio.git
cd portfolio

# Serve locally
npx http-server -p 3000
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📁 Project Structure

```
Portfolio/
├── index.html          # Main page
├── styles.css          # Custom CSS (glassmorphism, marquee, etc.)
├── script.js           # Scroll animation & interaction logic
├── assets/
│   └── frames/         # 240 JPEG frames for hero animation (Git LFS)
├── .gitignore
└── README.md
```
