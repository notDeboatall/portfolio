Design System — Debojeet Portfolio
1. Design Philosophy

A luxury minimalist developer portfolio inspired by Apple, Linear, and Vercel. The design focuses on clarity, whitespace, typography hierarchy, and subtle motion instead of visual noise.

The goal is to communicate:

High-end developer brand
AI + Full-stack capability
Clean engineering mindset
2. Color System

Base Colors

Background: #0A0A0A
Surface Dark: #111111
Card Surface: #171717

Text

Primary: #FFFFFF
Secondary: #A1A1AA
Muted: #71717A

Accent

Luxury Gold: #D4AF37

Borders

rgba(255,255,255,0.08)
3. Typography
Headings: Manrope / General Sans (SemiBold – Bold)
Body: Inter (Regular – Medium)

Hierarchy:

H1: 64–80px
H2: 40–48px
H3: 24–28px
Body: 16–18px
4. UI Style Rules
Border radius: 16–24px
Minimal shadows only (soft blur, low opacity)
No heavy gradients
No neon colors
Smooth hover transitions (200–300ms)
Layout grid: 12-column system
Large whitespace priority
5. Component Style

Buttons

Solid black + gold border hover
Subtle glow on hover

Cards

Dark surface with thin border
Lift effect on hover (translateY -4px)

Navigation

Sticky glass-like dark bar (very subtle blur)
